# Module 3: The AI-Robot Brain for Humanoid Robotics

This directory contains the complete Module 3 for the Physical AI & Humanoid Robotics course.

## Files

- `index.md` - Module overview and introduction
- `chapter-1-simulation.md` - Photorealistic simulation and synthetic data generation
- `chapter-2-vslam-navigation.md` - Hardware-accelerated VSLAM and navigation
- `chapter-3-path-planning.md` - Path planning and bipedal humanoid movement
- `glossary.md` - Key terms and definitions
- `troubleshooting.md` - Common issues and solutions
- `assessment.md` - Module-wide assessment questions
- `accessibility-statement.md` - Accessibility information
- `image-optimization.md` - Guidelines for image optimization

## Running the Documentation

To run this documentation locally:

1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm run start`
4. Open your browser to `http://localhost:3000`

The Module 3 content will be available at `/docs/modules/module-3/`.

## Deployment

To build the static site for deployment:

1. Run: `npm run build`
2. The output will be in the `build/` directory
3. Deploy the contents to your web server or GitHub Pages
