This document outlines the consistent frontmatter structure to be used across all module pages:

```markdown
---
title: [Page Title]
sidebar_label: [Sidebar Label]
description: [Brief description of the page content]
tags: [relevant, tags, for, the, content]
---
```

For chapter pages, the frontmatter should include:

```markdown
---
title: [Chapter Title]
sidebar_label: [Short Chapter Label]
description: [Brief description of what students will learn in this chapter]
tags: [robotics, ai, [specific-topic], tutorial, hands-on]
learning_objectives:
  - [Objective 1]
  - [Objective 2]
  - [Objective 3]
---
```
