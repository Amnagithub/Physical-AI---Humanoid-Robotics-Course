# Accessibility Statement for Module 3: The AI-Robot Brain for Humanoid Robotics

## Our Commitment

We are committed to making this educational content accessible to all learners, including those with disabilities. This module follows Web Content Accessibility Guidelines (WCAG) 2.1 AA standards to ensure an inclusive learning experience.

## Accessibility Features

- **Alternative Text**: All images and diagrams include descriptive alternative text
- **Semantic Structure**: Content uses proper heading hierarchy (H1, H2, H3, etc.)
- **Keyboard Navigation**: All interactive elements are accessible via keyboard
- **Screen Reader Support**: Content is structured for optimal screen reader experience
- **Color Contrast**: All text meets minimum contrast ratios for readability
- **Plain Language**: Technical concepts explained in accessible language

## Content Structure

This module uses the following accessibility-compliant structure:

### Headings
- H1: Page title (unique per page)
- H2: Major sections
- H3: Subsections
- H4: Detailed subsections

### Code Blocks
All code examples include:
- Syntax highlighting
- Language specification
- Descriptive captions when needed

### Lists
- Ordered lists for sequential information
- Unordered lists for related items
- Proper nesting structure

## Navigation Aids

- Clear breadcrumbs showing location within the module
- Consistent sidebar navigation
- "Related Concepts" sections linking to other chapters
- Glossary for technical terms

## Technical Requirements

To access this content with optimal accessibility:
- Use a modern web browser (Chrome, Firefox, Safari, Edge)
- Ensure JavaScript is enabled for interactive elements
- Use browser zoom features rather than page zoom for best results
- Screen readers: JAWS, NVDA, or VoiceOver recommended
